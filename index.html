Операційна система GNU/Linux

Коротка інформація

GNU/Linux — це родина операційних систем, заснованих на ядрі Linux та програмному забезпеченні GNU. Ці системи є відкритими та безкоштовними, що робить їх популярними для серверів, десктопів та вбудованих систем.

Популярні дистрибутиви

  • Ubuntu
  • Fedora
  • Debian
  • Arch Linux
  • CentOS
  • OpenSUSE

Приклади команд CLI

  • pwd - виводить поточний каталог
  • echo "Hello, World!" - виводить текст на екран
  • ls - показує вміст каталогу
  • cd - змінює каталог
  • touch файл.txt - створює новий файл
  • mkdir новий_каталог - створює новий каталог
  • cp файл.txt новий_файл.txt - копіює файл
  • mv файл.txt новий_каталог/ - переміщує файл
  • rm файл.txt - видаляє файл
  • history - показує історію команд

BASH-скрипт авторизації

#!/bin/bash
echo "Введіть ваше ім'я користувача:"
read username
echo "Введіть пароль:"
read -s password
echo "Авторизація успішна для $username"
